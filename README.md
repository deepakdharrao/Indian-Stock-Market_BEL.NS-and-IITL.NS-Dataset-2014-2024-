# Indian-Stock-Market_BEL.NS-and-IITL.NS-Dataset-2014-2024-
This repository provides daily historical stock market data for BEL.NS (Bharat Electronics Limited) and IITL.NS (Indian Infotech and Software Limited) listed on NSE, India, from 2014 to 2024. The dataset includes OHLC prices, adjusted close, and trading volume, suitable for time-series analysis and ML research.
